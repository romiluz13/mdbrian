# @memongo/lib

Shared Memongo runtime utilities and types used by the published engine and bridge packages.

This package exists to keep the public package dependency graph installable. It is not a primary integration surface and most consumers should use one of these packages instead:

- `@memongo/memory-engine`
- `@memongo/memory-bridge`
- `@memongo/memory`
- `@memongo/client`
- `@memongo/tools`
