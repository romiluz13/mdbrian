# @mdbrain/lib

Shared Mdbrain runtime utilities and types used by the published engine and bridge packages.

This package exists to keep the public package dependency graph installable. It is not a primary integration surface and most consumers should use one of these packages instead:

- `@mdbrain/memory-engine`
- `@mdbrain/memory-bridge`
- `@mdbrain/memory`
- `@mdbrain/client`
- `@mdbrain/tools`
